 Overview
This is a fully **JavaScript-based Portfolio Operating System**, showcasing my skills, projects, and personal info in an interactive desktop-like environment.  
It features **movable windows, a dock, apps, and theme switching**, all implemented using pure JavaScript, HTML, and CSS.



 Features
- About Me App: Displays personal information and role.
- Skills App: Shows a list of professional skills.
- Projects App: Highlights key projects with descriptions.
- Terminal App: Interactive terminal simulation with typing effect.
- Settings App: Toggle between light and dark themes.
- Dock:** Quick access to all apps.
- Draggable Windows: All windows can be moved freely on the desktop.


Technologies Used
- HTML5  
- CSS3 (via JavaScript injection)  
- JavaScript (ES6+)
